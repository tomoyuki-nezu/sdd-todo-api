## 背景・目的

新規プロジェクトの初期化に必要な決定事項を整理します。回答内容に基づいてプロジェクト構造・設定ファイル・CI/CD を自動生成します。

## 前提条件

- このプロジェクトは SDD（仕様書ドリブン開発）で進めます
- CI/CD は GitHub Actions を使用します
- ローカル開発環境は Docker を使用します
